# FreshMart Style Guide Notes

- `css/design-tokens.css`

## Design system summary

- Brand direction: premium fresh grocery, warm neutral background, deep forest green, amber accent.
- Primary UI font: `Be Vietnam Pro`.
- Editorial/brand font: `EB Garamond`.
- Main colors:
  - Background: `#f5f0e8`
  - Surface: `#ffffff`
  - Primary: `#0e3822`
  - Primary dark: `#082413`
  - Primary light: `#d4e8da`
  - Accent: `#b8772b`
  - Sale: `#a3291e`
  - Footer: `#0b2417`
  - Border: `#ddd7cd`

## Assumptions

- The report assets are clean reconstructed diagrams based on the current UI, not raw screenshots.
- Logo mark is simplified for documentation; use the real site logo asset when exporting production brand files.
- Font rendering may fall back locally if `Be Vietnam Pro` or `EB Garamond` is not installed, but the notes keep the correct design specification.
