## Component set

- Product Card Component: catalog/shop card pattern with image, badges, rating, price, wishlist, and add-to-cart action.
- AI Cooking Assistant Input Component: Meal Planner ingredient picker and filter controls.
- AI Suggested Dish Result Component: selected recipe result with meal status and ingredient availability.
- Missing Ingredient Product Link Component: Meal Planner to Shop mapping for missing ingredients.
- Cart Summary Component: cart/checkout summary with totals, discount, and checkout action.
- Admin Sidebar / Dashboard Components: admin navigation, KPI cards, product table, and actions.
